# alfred.romanizer

romanizes Japanese Katakana/Hiragana sentence.

## Requirements

* Alfred v2
* Alfred Powerpack license

## How to install

download via <http://kosugi.github.io/alfred.romanizer/>.

or clone this repository and make install (python 2, ImageMagick are required)
